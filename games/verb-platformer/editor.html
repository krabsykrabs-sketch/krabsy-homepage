Keyboard shortcuts

  • ↑↓←→ — pan camera
  • Scroll — zoom
  • Left-drag from an object (in select mode) — move it / the selection, snapped to the grid
  • Left-drag from empty ground (in select mode) — box-select objects
  • Ctrl+click — add/remove from current selection
  • Ctrl+A — select all · Ctrl+D — duplicate selection
  • F — frame selection (everything when nothing selected)
  • Middle-drag (or right-drag) — pan · right-drag in perspective orbits
  • WASD — nudge selected object (or hovering preview) by half a cell
  • PgUp / PgDn (or Shift+W/S) — move selected object up / down 1 unit (steps the Y plane while placing)
  • Q / E — rotate selected (or preview) by 45° (multi-select rotates around centroid)
  • R — rotate by 90° (Shift+R for 15°)
  • Enter — place hovering object at the preview position
  • P / O — perspective / top-down
  • + / - (or [ ]) — change Y plane
  • Esc / right-click — exit placement, return to select mode
  • Delete / Backspace — delete selected object
  • Ctrl+Z / Ctrl+Y — undo / redo
  • Ctrl+S — save JSON
  • L — toggle ID labels
  • G — toggle grid
  • ? — toggle this help

Click an object in the left panel to start placing. Right-click or Esc to stop.
